# Vision FM · Premium Live Radio Streaming Experience

Vision FM is a premium, high-fidelity web-based radio application designed for a seamless listening experience. Stream your favorite Indian and international radio stations with crystal-clear HD audio, an elegant glassmorphism UI, and a focus on performance.

## 📻 Key Features

*   **Premium HD Streaming:** Enjoy high-bitrate audio for a superior listening experience.
*   **Live Indian Radio Stations:** Access popular stations including Radio Mirchi, Vividh Bharati, Big FM, Red FM, and Air FM Gold.
*   **Diverse Genres:** From Bollywood hits and 90s classics to News, World Service, and Regional (Malayalam) content.
*   **Elegant Glassmorphism UI:** A modern, dark-themed interface built with Inter and Outfit typography.
*   **Ad-Free Experience:** Focus on the music with a clean, distraction-free environment.
*   **Cross-Platform Performance:** Fully responsive design optimized for mobile, tablet, and desktop browsers.
*   **Interactive Mini-Player:** Keep track of your stream with a persistent, intuitive playback control.

## 🎶 Featured Stations

Vision FM brings you the best of the airwaves:
- **Radio Mirchi 98.3 FM** (Bollywood & Hits)

## 🛠️ Built With

- **HTML5 & CSS3:** Utilizing modern CSS variables and Flexbox for a responsive layout.
- **JavaScript (Vanilla):** Light-weight and fast stream handling.
- **Google Fonts:** Inter & Outfit for premium typography.
- **Font Awesome:** For crisp, scalable iconography.
